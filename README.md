# Colox
This is my attempt at following the [Crafting Interpreters](https://craftinginterpreters.com/) book.

The implementation is in [Coalton](https://coalton-lang.github.io/).
> Coalton is an efficient, statically typed functional programming language that supercharges Common Lisp by taking great ideas from Haskell, Scheme, and OCaml.

This project is me scratching a few different itches at once:
- Never having done a project in a Lisp-like language. Why do people go crazy over Lisp?
- Wanting to play more with ML-derived languages.
- Getting back to my favorite CS course, and taking it a step further.
